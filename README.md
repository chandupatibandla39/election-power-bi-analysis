# Election Analysis Dashboard - Andhra Pradesh

**Author:** Purna Chandra Sekhar
GitHub: https://github.com/chandupatibandla39

## About This Project

This is a Power BI dashboard that analyzes the 2019 and 2024 Andhra Pradesh election results.
The data was downloaded from the Election Commission of India (ECI) official website.
The dashboard has 16 pages covering detailed analysis of vote share, constituency performance,
party comparison, voter turnout, and regional dominance.

## Data Source

- Election Commission of India (ECI)
- Andhra Pradesh Assembly Elections 2019
- Andhra Pradesh Assembly Elections 2024
- Official website: https://eci.gov.in

## Dashboard Pages

1. Vote Share Crisis Map
2. Constituency Rankings
3. Welfare Spend vs Vote Swing
4. Youth and Literacy Gap
5. Vote Share and Poll Volume
6. Party Efficiency Matrix
7. Seat Conversion Analysis
8. Vote Concentration Map
9. Close Contest Performance
10. Regional Dominance Zones
11. Turnout vs Performance
12. Party Growth 2014 to 2024
13. Opposition Comparison
14. Vote Share Comparison
15. Turnout Trends 2019 to 2024
16. Final Overview

## Tools Used

- Microsoft Power BI Desktop
- Power Query for data cleaning

- Data source: Election Commission of India
